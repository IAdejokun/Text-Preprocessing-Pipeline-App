Text Preprocessing Pipeline Application

A web application for text preprocessing using Django REST framework and React.

Features

- Tokenization
- Stemming
- Lemmatization
- Stopword removal

Requirements

- Python: 3.8+
- React: 18.3.1
- Django: 5.1.4
- Django REST framework: 3.15.2
- asgiref: 3.8.1
- sqlparse: 0.5.3
- typing_extensions: 4.12.2
- tzdata: 2024.2
